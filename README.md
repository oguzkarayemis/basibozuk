# başıbozuk e-yayıncılık
Bu depo, başıbozuk e-yayıncılık'ın IPFS yeraltı web sitesinin kaynak kodlarını barındırır. "Karanlıkta aranan bir ses" felsefesiyle, sıfır JS (Astro) mimarisi kullanılarak inşa edilmiştir.

## Mimari Özellikler
- **Framework:** Astro (Static Site Generation)
- **Barındırma:** IPFS (InterPlanetary File System) & ENS
- **Veritabanı:** Markdown tabanlı İçerik Koleksiyonları (Zod Type Safety)
- **Stil:** Vanilla CSS (Başıbozuk renk paleti: f7f4e7, 090f10, 404247 ve Merriweather lokal font)

## Geliştirici Kurulumu
1. Bağımlılıkları yükleyin: `npm install`
2. Yerel sunucuyu başlatın: `npm run dev`
3. IPFS için statik derleme: `npm run build`
